## üéµ Music Commands

| Command | Description |
| :--- | :--- |
| `play` | Plays a song from YouTube, Spotify, URL, etc. |
| `playnext` | Adds a song to the very top of the queue. |
| `playlocal` | Plays a local audio file from the host. |
| `nowplaying` | Shows the currently playing track. |
| `queue` | Displays the current song queue. |
| `skip` | Skips the current song. |
| `skipto` | Skips to a specific song number in the queue. |
| `stop` | Stops the music and clears the entire queue. |
| `pause` | Pauses the current playback. |
| `resume` | Resumes the paused song. |
| `volume` | Sets the player volume (0-200). |
| `seek` | Seeks to a specific time in the song (e.g., `1m 30s`). |
| `loop` | Toggles loop mode for the song or entire queue. |
| `autoplay` | Toggles autoplay mode for recommended tracks. |
| `shuffle` | Shuffles the current queue. |
| `grab` | Saves the current song to your Direct Messages. |
| `lyrics` | Display lyrics for the current track. |
| `join` | Joins the bot to your current voice channel. |
| `leave` | Disconnects the bot from the voice channel. |
| `clearqueue` | Clears all songs from the queue. |
| `remove` | Removes a specific song from the queue. |
| `replay` | Replays the current track from the beginning. |
| `search` | Search for a specific song and choose from results. |
| `move` | Move a specific track's position in the queue. |

## ‚ú® Filter Commands

| Command | Description |
| :--- | :--- |
| `8d` | Toggle the immersive 8D audio filter. |
| `bassboost` | Toggle the bassboost filter (Levels: low, medium, high). |
| `distortion` | Toggle the distortion audio filter. |
| `karaoke` | Toggle the karaoke filter for vocal removal. |
| `lowpass` | Toggle the lowpass filter for a muffled sound. |
| `nightcore` | Toggle the high-tempo nightcore filter. |
| `pitch` | Adjust the pitch of the music (0.5 - 5). |
| `rate` | Change the playback rate of the song. |
| `speed` | Change the speed of the song playback. |
| `tremolo` | Toggle the tremolo (wobbly volume) filter. |
| `vibrato` | Toggle the vibrato (wobbly pitch) filter. |
| `rotation` | Toggle the rotation (panning) filter. |
| `reset` | Resets and disables all active audio filters. |
